\begin{frame}
\frametitle{Objectives}
\label{sec-2}

At the end of this tutorial, you will be able to,
\begin{itemize}
\item Remove files and directories
\item Copy files from one location to another
\item Move files and directories
\item Know the Linux File Hierarchy
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Pre-requisite}
\label{sec-3}

Spoken tutorial on, 
\begin{itemize}
\item Getting started with Linux
\end{itemize}
\end{frame}


\begin{frame}
  \frametitle{Linux File Hierarchy}
  \begin{itemize}
  \item \texttt{/} is the root directory
  \item The root directory is the topmost level of the hierarchy
  \item For details \texttt{\em \$man hier}
  \end{itemize}
\end{frame}


\begin{frame}
\frametitle{Summary}
\label{sec-8}

  In this tutorial, we have learnt to,


\begin{itemize}
\item Remove files using ``rm'' command
\item Copy and move files from one location to another, using the ``cp'' 
      and ``mv'' commands respectively
\item Learnt the file system hierarchy of Linux
\end{itemize}
\end{frame}


\begin{frame}[fragile]
\frametitle{Evaluation}
\label{sec-9}


\begin{enumerate}
\item How to copy all the contents of one folder into another?
\vspace{15pt}
\item How will you rename the file \texttt{\em wonderland.txt} to \texttt{\em alice.txt} using the 
      commands learnt so far? 
\end{enumerate}
\end{frame}
\begin{frame}
\frametitle{Solutions}
\label{sec-10}


\begin{enumerate}
\item \$ cp folder1/* folder2 
\vspace{15pt}
\item \$ mv wonderland.txt alice.txt
\end{enumerate}
\end{frame}
